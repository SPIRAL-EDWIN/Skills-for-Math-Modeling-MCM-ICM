# MCM/ICM Modeling Skills

A clean, public, MIT-licensed export of Chen Jing's mathematical modeling skills for MCM/ICM-style workflows.

The reusable skill definitions live in [`.github/skills/`](.github/skills/). They cover data cleaning, evaluation models, forecasting, differential equations, graph theory, optimization, uncertainty analysis, visualization, LaTeX writing, PDF handling, spreadsheet workflows, and AI-use reporting for COMAP competitions.

## What This Repository Contains

- 42 reusable skill definitions in `.github/skills/`
- Source-code-style examples, notebooks, scripts, and Markdown documentation
- A professional English master guide: [`.github/skills/README.md`](.github/skills/README.md)
- A concise skill index: [`SKILLS_INDEX.md`](SKILLS_INDEX.md)
- A third-party and excluded-file notice: [`THIRD_PARTY_AND_EXCLUDED_FILES.md`](THIRD_PARTY_AND_EXCLUDED_FILES.md)
